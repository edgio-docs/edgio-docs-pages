createApolloMiddleware

  • createApolloMiddleware(baseURL?: string): ApolloLink

  • Creates an Apollo middleware that appends ?key=sha256(query + variables) to all http requests so that they can be cached by the URL alone.

    Example Configuration

     import createApolloMiddleware from '@xdn/prefetch/apollo/createApolloMiddleware'

 export default function(context) {
   return {
     link: createApolloMiddleware(process.env.SHOPIFY_GQL_ENDPOINT),
     httpEndpoint: process.env.GQL_ENDPOINT,
     httpLinkOptions: {
       credentials: 'omit',
       headers: { 'Access-Token': process.env.GQL_ACCESS_TOKEN }
     }
   };
 }

    Parameters

    • Default value baseURL: string = "/graphql"

      The URL for the graphql API endpoint. Defaults to '/graphql'.

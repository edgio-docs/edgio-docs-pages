# Getting Started

## Creating a new XDN app

To create a new XDN app, run:

```bash
npm create xdn-app@latest
```

... or using yarn ...

```bash
yarn create xdn-app@latest
```

## Adding the XDN to an existing app

To run an existing app on the XDN, run the following:

```bash
npm i -g @xdn/cli
npm init
```
